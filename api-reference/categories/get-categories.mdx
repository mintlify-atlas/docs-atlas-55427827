---
title: 'Get categories'
description: 'Retrieve the list of all transaction categories.'
api: 'GET /api/categories'
---

Returns an array of all categories available for tagging transactions.

<Note>
  This endpoint takes no query parameters. All categories are returned without pagination.
</Note>

## Response

<ResponseField name="" type="Category[]">
  Array of category objects.

  <Expandable title="Category properties" defaultOpen={true}>
    <ResponseField name="id" type="string">
      Unique identifier for the category, assigned by the server.
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Display name of the category.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url "${VITE_API_URL}/api/categories"
```

</RequestExample>

<ResponseExample>

```json 200
[
  {
    "id": "1",
    "name": "Food"
  },
  {
    "id": "2",
    "name": "Transport"
  },
  {
    "id": "3",
    "name": "Salary"
  }
]
```

</ResponseExample>
